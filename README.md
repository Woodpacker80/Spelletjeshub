# Rad van Woorden

Een responsief, browser-gebaseerd woordspel geïnspireerd op het klassieke tv-format — draai het rad, raad medeklinkers, koop klinkers en los de puzzel op voordat de computer dat doet.

## Functies

- 🎡 Geanimeerd draaiend rad met Web Audio geluidsmotor
- 🤖 AI-computeropponent (Makkelijk / Gemiddeld / Moeilijk)
- 📱 Volledig responsief — mobiel, tablet en desktop
- 🔊 Dempenknop om alle geluiden uit te schakelen
- ⭐ Bonusrondes met Chroom Bonusvak (+€2.000) en Min-vak (−50% rondeverdiensten)
- 🎉 Ballon- & confettibedankscherm

## Bonusronde Regels

| Spellengte  | Bonusrondes  |
|-------------|--------------|
| 3 rondes    | Laatste 1 ronde |
| 5 rondes    | Laatste 2 rondes |
| 7 rondes    | Laatste 2 rondes |

**Chroom Bonusvak** — land erop, raad een goede medeklinker en verdien je letterwaarde **plus €2.000 extra**. Een spanningspauze van 1,8 seconden met naafpuls en shimmergeluid speelt af voordat het toetsenbord activeert.

**Min-vak** — land erop en verlies **de helft van je huidige rondeverdiensten**.

## Lokaal Spelen

1. Download alle bestanden in één map
2. Open `index.html` in een moderne browser
3. Geen bouwstap of server vereist

## GitHub Pages Implementatie

1. Push alle bestanden naar een GitHub-repository
2. Ga naar **Settings → Pages**
3. Stel de bron in op `main` branch, root map
4. Je spel is live op `https://jouwgebruikersnaam.github.io/jouw-repo-naam/`
